# 文字动画生成器 v1.7 使用说明

## 一、运行环境要求
- Python 3.7 或更高版本
- 操作系统：Windows / macOS / Linux（推荐 Windows 以获得最佳字体支持）

## 二、依赖库安装（pip）
打开命令行（终端），执行以下命令安装所需第三方库：

```bash
pip install pygame jieba
```

`tkinter` 为 Python 内置库，无需额外安装。  
若系统提示 `tkinter` 缺失（常见于 Linux），请根据发行版安装相应包，例如 Ubuntu：
```bash
sudo apt-get install python3-tk
```

## 三、首次运行
1. 将程序文件（例如 `文字动画生成器.py`）放置于任意目录。
2. 在该目录下打开命令行，运行：
   ```bash
   python 文字动画生成器.py
   ```
3. 程序启动后会自动生成 `config.json` 配置文件，并显示主窗口。

## 四、主界面与右键菜单
启动后窗口显示为**纯色背景**，所有功能通过**鼠标右键**调出菜单操作。

- **右键菜单选项**：
  - **打开设置**：打开/关闭设置面板。
  - **开始**：开始播放文字动画。
  - **暂停**：暂停动画。
  - **清除**：清除屏幕上所有文字。
  - **均匀分布: 开/关**：切换文字起始位置是否按网格对齐。
  - **退出**：保存配置并退出程序。

## 五、设置面板详解
右键选择“打开设置”后，会弹出浮动设置窗口。所有修改**实时生效**，但只有点击底部“保存设置”才会写入 `config.json`。

### 1. 文件管理
- 支持添加最多 **10 个 .txt 文本文件**。
- **浏览文件**：通过系统对话框选择文件。
- **清空文件**：移除所有已载入的文件。
- **支持拖拽**：直接将 .txt 文件拖入主窗口即可添加。
- 已添加的文件会显示在列表中，**单击文件名可移除**该文件。
- 多个文件的内容会被合并，统一分词后作为文字来源。

### 2. 运动速度
- **固定速度**：所有文字以相同速度移动。
- **随机速度**：每个文字的速度在“基础速度”的一定百分比范围内随机波动。
- **基础速度**：1~20，数值越大移动越快。
- **随机百分比**：0%~200%，控制速度的随机变化幅度。

### 3. 文字大小
- **固定大小**：所有文字字号一致。
- **随机大小**：每个文字的字号在“基础大小”的百分比范围内随机。
- **基础大小**：12~120。
- **随机百分比**：0%~200%。

### 4. Token 生成速度
- 每秒生成文字的数量，范围 50~3000。实际生成量受帧率限制，设置过高时会达到性能上限。

### 5. 动画方式（可多选）
- **降落**：文字从顶部下落。
- **上升**：文字从底部上升。
- **左入弹幕**：文字从右侧水平飞入。
- **右入弹幕**：文字从左侧水平飞入。
- **中心炸开**：文字从屏幕中心向四周散开。
- **四周聚拢**：文字从屏幕四周向中心汇聚（到达中心后渐隐消失）。
- 至少需勾选一种，否则不会生成文字。

### 6. 文字透明度
- **固定透明度**：所有文字透明度一致。
- **随机透明度**：每个文字的透明度在基础透明度的一定百分比内随机。
- **基础透明度**：50~255（255 为完全不透明）。
- **随机百分比**：0%~200%。

### 7. 翻转与旋转
- **随机水平翻转**：每个文字有 50% 概率左右镜像。
- **随机垂直翻转**：每个文字有 50% 概率上下颠倒。
- **启用随机旋转**：每个文字随机旋转一定角度。
- **最大旋转角度**：0~360，控制随机旋转的最大角度。

### 8. 字体选择
- 内置字体：黑体、微软雅黑、宋体、楷体、仿宋。

### 9. 文字颜色模式
- **每个 token 随机颜色**：每个文字的颜色完全随机。
- **整体统一颜色**：所有文字使用同一种颜色（可从预设中选择）。

### 10. 背景颜色
- 点击“选择背景颜色”弹出 12 色网格窗口，点击色块即可切换背景。

### 11. 字符筛选
- 控制分词后哪些类型的字符可以出现在动画中。
- **符号**：中英文标点符号。
- **数字**：0-9。
- **大写字母**：A-Z。
- **小写字母**：a-z。
- **中文**：汉字（Unicode 范围）。
- 当**未载入任何文件且未设置自定义字符**时，勾选的选项将生成对应的预设字符集（符号为常用中英文符号，中文为“〇壹贰叁肆伍陆柒捌玖”）。

### 12. 字符长度过滤
- 只允许包含至少 N 个筛选后字符的 token 出现，范围 1~10。

### 13. 自选字符
- 点击“编辑自选字符”会弹出**系统原生输入框**。
- 输入内容以**英文逗号**分隔，如：`你好,世界,Hello`。
- **重要规则**：
  - 当自选字符框**不为空**时，将**直接使用这些内容**作为 token，完全无视“字符筛选”和载入的文件。
  - 若为空，则按文件内容或预设字符集生成 token。

### 14. 保存与关闭
- **保存设置**：将当前所有面板配置写入 `config.json`，下次启动自动加载。
- **关闭窗口**：关闭设置面板（不会退出程序）。

## 六、窗口缩放
主窗口支持**自由拉伸**（鼠标拖动边框或最大化），设置面板大小固定，不会随主窗口缩放。

## 八、常见问题
**Q：为什么设置了每秒生成 3000，实际感觉没有那么多？**  
A：受限于电脑性能和帧率，实际每秒生成数量可能无法达到极高设定值。建议设置在 200~500 之间即可获得密集效果。

**Q：自定义字符和文件同时存在时会怎样？**  
A：自定义字符优先，会忽略载入的文件和字符筛选。清空自定义字符后，文件或预设字符集才会生效。

**Q：如何获得更多字体？**  
A：程序使用系统已安装字体。将 .ttf 字体文件安装到系统后，可以在代码的字体映射中添加对应的名称。如需指导可参考程序注释。

**Q：动画卡顿怎么办？**  
A：可适当降低“每秒生成数量”，或减少动画方式勾选数量，避免同时存在过多文字元素。
